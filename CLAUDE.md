# entri

AI study-notes app: photo capture → Claude vision extraction (never silently corrects) →
FSRS daily exam tuned to exam date → grounded RAG chat. Thesis: **retention + trust**.
See `PLAN.md` for the full CEO/eng plan and milestones (M0–M6).

## Design System
Always read `DESIGN.md` before making any visual or UI decisions.
All font choices, colors, spacing, and aesthetic direction are defined there.
The direction is **Marginalia (dialed back)** — a warm notebook, taken seriously; the
signature move is the marigold highlighter swipe, and AI-inferred content must always
render visibly tentative (taupe / dashed / sticky), never disguised as confirmed.
Do not deviate without explicit user approval.
In QA mode, flag any code that doesn't match DESIGN.md.
